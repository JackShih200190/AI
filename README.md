# AI Agent 
AI Agent 的核心功能包括:
  1. 規劃 - 將目標拆解為小步驟並逐一執行,同時進行自我反思與學習
  2. 記憶 - 儲存並檢索資訊以改進行動
  3. 工具使用 - 能夠呼叫API或使用Python函式等工具

## Workflow 
你可以有一個約時間的AI、一個確認時間的AI、一個寫訪綱的AI⋯⋯.，這個工作流程被稱為代理工作流程（Agentic Workflow），重點在於上述提到的：計畫與記憶能夠修正、使用的工具與行動能夠調整。
